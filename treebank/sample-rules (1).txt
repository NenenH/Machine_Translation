Syntantic Rules 

Lexical Phrase
DP [Determiner]
PP [Preprosition]
VP [Verb]
AP [Adjective]
NP [Noun]

QP [Quantifier]


DP = D' + QP | D' + AP
D' = D + NP

PP = P' + ADV | P' + TP
P' = P + DP

AP = A' + ADV
A' = A + DP

VP = V' + ADV | V' + DP
V' = V + DP ¦ V + PRN ¦ V' + CP

NP = N' + attribute phrase
N' = N + PP



TP [Tense Phrase]
TP = T' + DP | T' + QP
